# 게임 정체성 및 차별화 전략

## 개요

이 문서는 "로그라이크 덱빌딩 야구 게임"이 Balatro와 어떻게 차별화될 수 있는지에 대한 분석과 제안을 정리합니다.

---

## 1. 현재 상황 분석

### 1.1 이미 Balatro와 다른 점

| 요소 | Balatro | 우리 게임 |
|------|---------|----------|
| 덱 시스템 | 단일 덱 (포커 카드) | **이중 덱** (선수덱 + 액션덱) |
| 핵심 자원 | 칩/배율 = 점수 | 선수 타율 + 족보 = **안타 확률** |
| 결과 결정 | 족보 = 확정 점수 | 족보 = **확률 기반 판정** |
| 진행 방식 | 블라인드 단위 | **이닝 단위** + 아웃 카운트 |
| 상태 누적 | 없음 | **베이스 러닝** (주자 진루/득점) |
| 덱 순서 | 무작위 셔플 | **예측 가능한 순환** (타순) |

### 1.2 그럼에도 "Balatro처럼" 느껴지는 이유

**문제점**: 플레이어의 주의력이 **"족보 만들기"**에 집중됨

- 선수 카드는 단순 타율 보정 역할
- 야구 요소가 "스킨"으로 느껴짐
- 핵심 의사결정: "어떤 족보를 만들까?" (= Balatro와 동일)

---

## 2. 차별화의 핵심 원칙

### 2.1 게임 정체성 정의

> **"9이닝의 서사 속에서 타순을 설계하고, 상황을 읽고, 팀을 운영하는 야구 감독 시뮬레이션 + 덱빌딩"**

### 2.2 핵심 질문 전환

| Balatro | 우리 게임 |
|---------|----------|
| "어떤 족보를 만들까?" | **"이 선수로, 이 상황에서, 어떻게 플레이할까?"** |

### 2.3 차별화 3대 축

1. **선수 중심**: 선수가 게임의 주인공
2. **타순 전략**: 덱 순서 설계가 핵심 게임플레이
3. **상황 인식**: 같은 카드도 상황에 따라 가치 변동

---

## 3. 핵심 차별화 요소

### 3.1 선수가 게임의 주인공

**현재**: 선수 = 타율 제공 (수동적)
**목표**: 선수 = 게임플레이 결정 (능동적)

#### 선수 고유 능력 시스템

```typescript
type PlayerAbility =
  | { type: 'clutch'; effect: '2사 상황에서 안타 확률 +20%' }
  | { type: 'leadoff'; effect: '이닝 첫 타석 출루 시 진루 +1' }
  | { type: 'cleanup'; effect: '주자 있을 때 장타 확률 +15%' }
  | { type: 'speedster'; effect: '1루타 후 자동 도루 시도' }
  | { type: 'run_producer'; effect: '타점 획득 시 포인트 1.5배' }
  | { type: 'wall_scraper'; effect: '아웃 판정 시 25% 확률로 희생타 전환' };
```

**효과**: "이 상황에 어떤 선수를 내보낼까"가 핵심 의사결정이 됨

---

### 3.2 타순 전략 (예측 가능한 순환)

**다른 덱빌딩과의 차이**:
- Slay the Spire: 덱 순서 모름, 셔플 기반
- Balatro: 핸드 단위 최적화
- **우리 게임**: 덱 전체 순서를 보고 "흐름"을 설계

#### 타순 시너지 시스템

| 시너지 | 조건 | 효과 |
|--------|------|------|
| 1-2 콤보 | 스피드 → 컨택 연속 타석 | 뒷 타자 안타 확률 +5% |
| 클린업 연계 | 파워 선수 2연속 | 두 번째 타자 장타 확률 +10% |
| 테이블 세터 | 선구안 선수 뒤 타석 | 볼 카운트 유리 시작 |

#### 타순별 역할

```
1번: 출루 특화 (선구안 보너스)
2번: 진루 특화 (번트/히트앤런 가능)
3번: 밸런스
4번: 클린업 (주자 있을 때 파워 보너스)
5-9번: 상황별 특화
```

---

### 3.3 상황에 따른 가치 변동

**핵심 개념**: 같은 "안타"도 상황에 따라 가치가 완전히 다름

#### 상황 보너스 시스템

| 상황 | 효과 |
|------|------|
| 2사 만루 | 모든 안타가 "결정타" 보너스 (+50% 포인트) |
| 9회 말 동점 | "끝내기" 가능 (홈런 시 특별 보상) |
| 무사 1루 | 번트 옵션 활성화 |
| 득점권 (2루/3루) | 타점 보너스 |
| 후반 이닝 (7-9회) | 모든 득점 포인트 1.2배 |

#### 아웃카운트 = 리소스

| 아웃 | 전략적 의미 |
|------|------------|
| 0아웃 | 희생번트로 주자 진루 가능 |
| 1아웃 | 밸런스 있는 선택 |
| 2아웃 | 올인 or 포기 |

**"실패를 전략적으로 사용"**:
- 번트로 아웃을 "소비"해 주자 진루
- 희생플라이로 1아웃을 "교환"해 1점 획득
- 지는 것도 이기는 전략의 일부

## 4. 확률 시스템 개선

### 4.1 현재 문제

- 확률 판정이 "운"으로 느껴짐
- 좋은 족보를 만들어도 아웃될 수 있음
- 덱빌딩 보상이 불확실

### 4.2 개선 방향: 최소 결과 보장 + 업그레이드 확률

```
현재: 족보 → 확률 판정 → 안타 or 아웃
제안: 족보 → 기본 결과 확정 + 확률로 업그레이드 기회
```

#### 티어별 결과 체계

| 족보 티어 | 기본 결과 (확정) | 업그레이드 확률 |
|----------|-----------------|----------------|
| Low (하이카드~투페어) | 확률 판정 유지 | - |
| Mid (트리플~플러시) | 1루타 확정 | 타율 + 보너스로 장타 |
| High (풀하우스 이상) | 결과 확정 | - |

### 4.3 Lucky Hit 연출 강화

```typescript
if (wasLucky) {
  // 낮은 확률로 성공할수록 큰 보너스
  bonusPoints = (1.0 - hitProbability) * 100;
  // 연출: 슬로우모션 + 특수 이펙트
  // 텍스트: "CLUTCH!", "MIRACLE!"
}
```

## 5. 선수덱 시스템 확장

### 5.1 선수 컨디션 시스템

```
컨디션 (0-100)
- 게임 시작: 50~80 랜덤
- 안타 시: +5~10 (자신감)
- 아웃 시: -5~15 (슬럼프)
- 홈런 시: +20, 팀 전체 +5

컨디션 효과:
- 80+: "핫 스트릭" 타율 +0.03
- 40-: "슬럼프" 타율 -0.05
```

### 5.2 선수 피로도 시스템

```
피로도 증가:
- 매 타석: +10
- 출루 대기: +5/타석
- 홈인 (전력 질주): +15

피로도 효과:
- 50+: 스피드 -1
- 70+: 타율 -0.02
- 90+: 모든 스탯 -20%

회복:
- 벤치 휴식 (덱 하단 대기)
- 상점 아이템
```

### 5.3 선수 케미스트리

| 케미스트리 | 조건 | 효과 |
|-----------|------|------|
| 배터리 메이트 | 같은 팀 출신 연속 타석 | 양쪽 타율 +0.02 |
| 라이벌 | 특정 선수 조합 | 둘 다 파워 +1 |
| 사제지간 | 베테랑 + 신인 | 신인 경험치 2배 |

## 6. 야구만의 고유한 경험

### 6.1 다른 덱빌딩에서 불가능한 것

| 요소 | 일반 덱빌딩 | 야구 덱빌딩 |
|------|------------|------------|
| 덱 순서 | 무작위 셔플 | **예측 가능한 순환** |
| 카드 가치 | 절대적 | **상황에 따라 변동** |
| 실패 | 피해야 할 것 | **전략적 도구 (희생)** |
| 리소스 | 마나/에너지 | **아웃카운트** |
| 연속성 | 턴 간 독립 | **주자 누적 → 득점** |
| 타이밍 | 언제든 강카드 | **후반 역전의 드라마** |

### 6.2 야구 게임 레퍼런스

| 게임 | 핵심 재미 | 적용 가능 요소 |
|------|----------|---------------|
| MLB The Show | 투수-타자 대결 | 읽기 싸움 미니게임 |
| 파워프로 | 선수 육성 | 선수 성장 시스템 |
| 야구 일반 | 9회 역전 | 후반 이닝 보너스 |

## 8. 검증 기준

### "Balatro 스킨"이 아닌지 확인하는 질문

1. ✅ 선수 선택이 "누가 타율 높지?" 이상의 의미가 있는가?
2. ✅ 덱 순서를 아는 것이 전략에 영향을 주는가?
3. ✅ 같은 족보라도 상황에 따라 플레이가 달라지는가?
4. ✅ "야구를 하고 있다"는 느낌이 드는가?
5. ✅ 9이닝의 흐름과 역전의 드라마가 있는가?
