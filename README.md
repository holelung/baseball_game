# 야구 덱빌딩 프로토타입

로그라이크 덱빌딩 야구 게임의 핵심 메커니즘을 검증하기 위한 프로토타입입니다.

## 핵심 규칙

- **이중 덱 시스템**: 선수덱 + 액션덱
- **선수덱**: 9장, 3장 드로우 후 1장 선택. 출루 시 루상, 아웃 시 덱 하단
- **액션덱**: 이닝마다 셔플, 안타/아웃 등의 결과 결정
- **이닝**: 3아웃 시 종료

## 기술 스택

- React + TypeScript
- Vite
- Zustand (상태 관리)
- Tailwind CSS

## 실행 방법

```bash
npm install
npm run dev
```

## 프로젝트 구조

```
src/
├── game/           # 순수 게임 로직 (Unity 포팅 대비)
├── store/          # Zustand 스토어
├── components/     # UI 컴포넌트
└── data/           # 스타터 카드 데이터
```
