# 액션 카드 족보 시스템

## 개요

스탯 기반 액션 카드 덱은 기존 트럼프 카드(52장)를 야구 테마로 리스킨한 시스템입니다.
기존 포커 족보의 익숙함을 유지하면서 야구 전용 족보를 추가하여 독창성을 확보합니다.

---

## 카드 구성

### 4가지 속성 (문양 대체)

| 속성 | 이모지 | 색상 | 설명 | 보너스 효과 |
|------|--------|------|------|-------------|
| **파워** | 💥 | 🔴 빨강 | 장타력을 상징 | 장타(2루타+) 확률 ↑, 타점 보너스 |
| **컨택** | 🎯 | 🔵 파랑 | 정확한 타격을 상징 | 안타 확률 ↑, 삼진 확률 ↓ |
| **스피드** | ⚡ | 🟢 초록 | 빠른 발을 상징 | 도루 성공률 ↑, 진루 보너스 |
| **선구안** | 👁️ | 🟡 금색 | 뛰어난 선구안을 상징 | 볼넷 확률 ↑, 투수 체력 소모 ↑ |

### 숫자 범위 (1-13)

| 범위 | 숫자 | 이름 | 효과 |
|------|------|------|------|
| **Low** | A, 2, 3, 4 | 컨택 유리 | 안타 확률 +5%, 장타 확률 -5% |
| **Mid** | 5, 6, 7, 8, 9 | 밸런스 | 추가 효과 없음 |
| **High** | 10, J, Q, K | 파워 유리 | 장타 확률 +5%, 안타 확률 -5% |

### 총 카드 수
- 4속성 × 13숫자 = **52장**

## 족보 상세 설명

### 기본 포커 족보

#### 하이카드 (High Card)
```
예시: 💥3  🎯7  ⚡K  👁️2  💥9
```
- 아무 조합도 완성되지 않은 상태
- 최소한의 점수만 획득

#### 원페어 (One Pair)
```
예시: 💥7  🎯7  ⚡3  👁️K  💥2
       └──┘
```
- 같은 숫자 카드 2장
- 가장 기본적인 족보

#### 투페어 (Two Pair)
```
예시: 💥7  🎯7  ⚡3  👁️3  💥K
       └──┘   └──┘
```
- 원페어가 2세트
- 안정적인 중급 족보

#### 트리플 (Three of a Kind)
```
예시: 💥7  🎯7  ⚡7  👁️K  💥2
       └────────┘
```
- 같은 숫자 카드 3장
- 풀하우스의 기반

#### 스트레이트 (Straight)
```
예시: 💥3  🎯4  ⚡5  👁️6  💥7
       └─────────────────┘ (연속)
```
- 연속된 숫자 5장 (속성 무관)
- A-2-3-4-5 또는 10-J-Q-K-A 가능

#### 플러시 (Flush)
```
예시: 💥2  💥5  💥8  💥J  💥K
       └─────────────────┘ (같은 속성)
```
- 같은 속성 카드 5장
- 속성 보너스 효과 활성화

#### 풀하우스 (Full House)
```
예시: 💥7  🎯7  ⚡7  👁️K  💥K
       └────────┘   └──┘
         트리플     페어
```
- 트리플 + 원페어 조합
- 강력한 상급 족보

#### 포카드 (Four of a Kind)
```
예시: 💥7  🎯7  ⚡7  👁️7  💥K
       └──────────────┘
```
- 같은 숫자 카드 4장
- 희귀하고 강력한 족보

#### 스트레이트 플러시 (Straight Flush)
```
예시: 💥3  💥4  💥5  💥6  💥7
       └─────────────────┘ (같은 속성 + 연속)
```
- 같은 속성으로 연속된 숫자 5장
- 최상위 기본 족보

### 야구 전용 족보

#### 배팅 아이 (Batting Eye) 👁️
```
예시: 👁️2  👁️5  👁️8  👁️J  💥3
       └────────────────┘ (선구안 4장)
```
**조건:** 선구안(👁️) 카드 4장 이상
**특수 효과:** 볼넷 획득 시 추가 포인트 +20

> 💡 선구안 카드를 모아서 투수를 지치게 하는 전략

#### 파워 서지 (Power Surge) 💥
```
예시: 💥K  💥Q  💥J  🎯2  ⚡3
       └──────────┘
       파워 3장, 합계 = 13+12+11+2+3 = 41 (≥30)
```
**조건:** 파워(💥) 카드 3장 이상 & 전체 숫자 합계 30 이상
**특수 효과:** 홈런 확률 2배

> 💡 높은 숫자 파워 카드를 모아 대포를 터트리는 전략

#### 스피드 스타 (Speed Star) ⚡
```
예시: ⚡3  ⚡4  ⚡5  ⚡6  ⚡7
       └─────────────────┘ (스피드로만 스트레이트)
```
**조건:** 스피드(⚡) 카드 5장으로 스트레이트 완성
**특수 효과:** 해당 타석 도루 자동 성공

> 💡 발이 빠른 선수와 조합하면 시너지 극대화

#### 컨택 마스터 (Contact Master) 🎯
```
예시: 🎯7  🎯7  🎯7  🎯K  🎯K
       └────────┘   └──┘ (컨택으로만 풀하우스)
```
**조건:** 컨택(🎯) 카드 5장으로 풀하우스 완성
**특수 효과:** 해당 타석 삼진 무효 (무조건 타구 발생)

> 💡 안타 제조기 타입의 선수에게 최적

#### 퍼펙트 스윙 (Perfect Swing) ✨
```
예시: 💥7  🎯7  ⚡5  👁️K  💥3
       └──┘
       4속성 각 1장 이상 포함 + 페어 보유
```
**조건:** 파워, 컨택, 스피드, 선구안 각 1장 이상 + 같은 숫자 2장
**특수 효과:** 결과 한 단계 업그레이드 (아웃→안타, 안타→2루타, ...)

> 💡 가장 어렵지만 가장 강력한 야구 전용 족보

## 점수 계산 공식

```
최종 포인트 = (기본 칩 + 카드 칩 합계) × (기본 배율 + 보너스 배율)
```

### 예시 계산

**상황:** 파워 서지 완성 (💥K, 💥Q, 💥J, 🎯5, ⚡3)

1. **기본 칩:** 50 (파워 서지)
2. **카드 칩 합계:** 13 + 12 + 11 + 5 + 3 = 44
3. **총 칩:** 50 + 44 = 94
4. **기본 배율:** ×6 (파워 서지)
5. **최종 포인트:** 94 × 6 = **564점**

## 속성 시너지

### 플러시 속성별 추가 효과

| 속성 플러시 | 추가 효과 |
|-------------|-----------|
| 💥 파워 플러시 | 장타 확률 +15% |
| 🎯 컨택 플러시 | 안타 확률 +15% |
| ⚡ 스피드 플러시 | 진루 +1 (1루타→2루타 효과) |
| 👁️ 선구안 플러시 | 투수 체력 2배 소모 |

### 선수 카드와의 시너지

선수 카드의 태그와 액션 카드의 속성이 일치하면 추가 보너스:

| 선수 태그 | 일치 속성 | 시너지 효과 |
|-----------|-----------|-------------|
| `power` | 💥 파워 | 장타 시 타점 +1 |
| `contact` | 🎯 컨택 | 안타 확률 +10% |
| `speed` | ⚡ 스피드 | 도루 성공률 +20% |
| `eye` | 👁️ 선구안 | 볼넷 확률 +15% |

## 전략 가이드

### 초반 (1-3이닝)
- **원페어/투페어** 위주로 안정적 출루
- 덱 순환시키며 카드 분포 파악

### 중반 (4-6이닝)
- **플러시/스트레이트** 도전
- 야구 전용 족보 조건 체크

### 후반 (7-9이닝)
- 목표 점수에 따라 **고위험 고수익** 또는 **안정적 득점** 선택
- **퍼펙트 스윙** 노리기
