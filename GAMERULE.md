# 게임 룰

## 개요
**목표**
9이닝동안 최대한 많은 투수들을 강판시키면서 목표 포인트를 획득하는 것이 목표

**투수** *(v0.2 예정)*
팀마다 특수한 옵션을 가진 투수들이 존재하여 플레이어들을 방해함

---

## 카드덱

### 선수 카드
선수카드는 총 10장을 가지고 시작하며 추후 상점에서 추가할 수 있음

- 10장의 카드는 셔플되어 게임 시작 시 **3장의 카드를 드로우**
- 선수 선택 후 나머지 2장은 덱 상단으로 복귀
- 선수 카드덱은 게임 시작(1회 시작) 이후 기본적으로 **셔플하지 않음**

**덱 순환 규칙:**
- 아웃된 선수: 덱 최하단으로 이동
- 득점한 선수: 덱 최하단으로 이동
- 루상 잔류 선수: 이닝 종료 시 덱 최하단으로 이동

> UI적으로 카드가 나올 순서를 표시하여, 플레이어가 다음 선수를 예측할 수 있어야 함

### 액션 카드 (트럼프 카드)
액션카드는 선수(타자)의 행동을 결정할 수 있는 카드

**기본 구성:** 52장의 트럼프 카드 (4무늬 × 13숫자)

카드의 조합(족보)을 통해 타자가 어떤 액션을 취하게 될 지 결정된다.
카드는 발라트로 방식으로 이후 추가하거나 특정 카드를 강화할 수 있다.

#### 족보 시스템 (v0.1.0 구현)
| 족보 | 조건 | 야구 결과 | 배율 |
|------|------|----------|------|
| 하이 카드 | 아무 조합 없음 | 아웃 | x1 |
| 원 페어 | 같은 숫자 2장 | 1루타 | x2 |
| 투 페어 | 페어 2개 | 1루타 | x3 |
| 트리플 | 같은 숫자 3장 | 2루타 | x4 |
| 스트레이트 | 연속된 5장 | 2루타 | x5 |
| 플러시 | 같은 무늬 5장 | 3루타 | x6 |
| 풀하우스 | 트리플 + 페어 | 3루타 | x7 |
| 포카드 | 같은 숫자 4장 | 홈런 | x8 |
| 스트레이트 플러시 | 연속 + 같은 무늬 | 홈런 | x10 |

## 보너스 카드 *(v0.2 이후 예정)*
보너스 카드는 게임 중 상점을 통해 구매할 수 있으며 종류는 다음과 같다

### 선수 관련
**선수 훈련**
기존에 가지고 있는 선수를 강화하여 점수 보너스 같은 특성을 추가할 수 있다

**선수 교체(영입)**
특정 능력을 가진 선수를 영입하여 기존의 선수와 교체할 수 있다.

### 액션 관련
**카드 강화**
카드 자체에 보너스를 줄 수 있다.

**족보 강화**
족보를 완성했을 때 얻는 보상(결과)를 강화 할 수 있다.

**카드 추가**
액션 카드덱에 새로운 카드를 추가 할 수 있다.

### 조커
조커 카드를 통해 게임 플레이 성향을 결정할 수 있는 여러 어드벤티지 혹은 어드벤티지 + 패널티를 가지고 있는 카드들이다.

### 바우처
상점이나, 게임 진행에 영향을 줄 수 있는 카드

## 점수제
게임 승리 방식은 **이닝별 목표 Point를 달성**하는 것이다.

### Point 계산 (v0.1.0)
```
Point = (기본 점수 × 족보 배율) + 득점 보너스
```

| 항목 | 설명 |
|------|------|
| 기본 점수 | 족보별 고정 점수 (10~100) |
| 족보 배율 | x1 ~ x10 |
| 득점 보너스 | 득점당 +20P |

### 목표 Point
- 1이닝: 100P
- 이후 이닝마다 +50P 증가 (2이닝: 150P, 3이닝: 200P ...)

## 액션 방식

### 현재 구현 (v0.1.0) - 확정형
족보에 따라 결과가 확정됨:
- 원 페어 이상 = 안타 (족보에 따라 1~4루타)
- 하이 카드 = 아웃

### 향후 개선 방향 (v0.2 예정) - 확률형
액션은 기본적으로 `선수 카드의 안타확률 + 액션카드 조합확률`이다.

플레이어는 이를 잘 조합하여 안타의 확률을 1 이상으로 만들 수 있으며, 다양한 조합을 통해 1루타 이상의 안타도 만들어 낼 수 있다.

> **카드 조합은 확률을 높혀주는 것이지 확정지어 주는 것이 아님**

## 액션카드 로테이션 방식

### 현재 구현 (v0.1.0)
- 매 타석 **5장 드로우**
- 선택된 카드로 족보 판정 (선택 안하면 5장 전체 사용)
- 이닝당 **2회 다시뽑기** 가능

### 향후 개선 방향 (v0.2 예정)
- 액션카드는 기본적으로 **8장**을 패에 들고 있음
- 플레이어는 제한된 버리기 횟수를 통해 한번에 **최대 5장까지 버리고** 새 카드를 뽑을 수 있음
- 선택한 카드 조합으로 족보를 구성하여 플레이
