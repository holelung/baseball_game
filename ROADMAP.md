### 확률 시스템
```
안타 확률 = 선수 타율 + 족보 hitBonus
```

| 족보 | hitBonus | 타율 0.300 기준 |
|------|----------|----------------|
| 하이 카드 | -50% | -20% (거의 아웃) |
| 원 페어 | +20% | 50% |
| 투 페어 | +40% | 70% |
| 트리플 | +50% | 80% |
| 스트레이트 | +60% | 90% |
| 플러시 | +70% | 100% (확정) |
| 풀하우스 | +80% | 100% (확정) |
| 포카드 | +100% | 100% (확정) |
| 스트레이트 플러시 | +150% | 100%+ (보너스) |

### Point 계산
```
Point = (기본 점수 × 족보 배율) + 득점 보너스 + 초과 확률 보너스
```
- 확률 1.0 초과 시 추가 보너스
