### 족보 → 야구 결과 매핑
| 족보 | 조건 | 야구 결과 | 배율 |
|------|------|----------|------|
| 하이 카드 | 아무 조합 없음 | 아웃 | x1 |
| 원 페어 | 같은 숫자 2장 | 1루타 | x2 |
| 투 페어 | 페어 2개 | 1루타 | x3 |
| 트리플 | 같은 숫자 3장 | 2루타 | x4 |
| 스트레이트 | 연속된 5장 | 2루타 | x5 |
| 플러시 | 같은 무늬 5장 | 3루타 | x6 |
| 풀하우스 | 트리플 + 페어 | 3루타 | x7 |
| 포카드 | 같은 숫자 4장 | 홈런 | x8 |
| 스트레이트 플러시 | 연속 + 같은 무늬 | 홈런 | x10 |

### Point 계산
```
Point = (기본 점수 × 족보 배율) + 득점 보너스
```
- 기본 점수: 족보별 고정 (10~100)
- 득점 보너스: 득점당 +20P
- 목표: 1이닝 100P, 이후 이닝마다 +50P

### 현재 제한 사항
- 5장 드로우 방식 (기획서의 8장 핸드와 다름)
- 족보에 따라 결과 확정 (기획서의 확률 기반과 다름)
- 이닝당 2회 다시뽑기
